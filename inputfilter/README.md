# wal_inputfilter

A set of `Input` validation structs used to validate primitive values as they pertain to web applications.

## Members

- `Input` - Rule struct to add validators, and/or 'filters' to.
- 'validators/'
  - `IntegerValidator`
  - `FloatValidator`
  - `StringValidator`
  - `EqualityValidator`
  - `ArrayValidator`
